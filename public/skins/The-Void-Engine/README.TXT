I originally decided to create a simple skin off of which
people (mostly myself) could base more complex
skins, an "Engine" (like a game engine) so to speak.
It contains almost nothing (just slightly more than the
bare requirements for usability), thus conjuring up the
fifth Buddhist element, the "Void".

The skin itself, only contains three colors: white for
the background, black for the foreground, and fifty
percent gray for a few times when a third color really
simplified things. The image files that make up the
skin actually contain two extra colors: neon pink for
the parts of the files that are unused filler space, and
aneurysm green (I kid you not) for the parts that
usually get covered by other widgets. The key word
in that last sentence is usually, there are some cases
that the green shows through, disturbing the harmony
of black and white. To compensate for this, I created
two versions of the skin: a "developer" version,
complete with nasty green, and a "production"
version without it. The production version is in the
main skin directory (so that if you equip the skin you'll
get the good looking one), and the developer version
is in a subdirectory.
